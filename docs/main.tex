\subsection{Χρήση Εργαλείων Τεχνητής Νοημοσύνης}
Επιλεγμένα σημεία της εργασίας υλοποιήθηκαν με τη βοήθεια εργαλείων τεχνητής νοημοσύνης. Συγκεκριμένα, χρησιμοποιήθηκε η δωρεάν έκδοση \eng{ChatGPT-5.2}, άνευ λογαριασμού χρήστη, στα ακόλουθα σημεία:

\begin{itemize}
\tightlist
\item
    Υλοποίηση \eng{triggers} για τις σχέσεις εποπτείας μεταξύ ιατρών.
\item
    Υλοποίηση \eng{triggers} για τους περιορισμούς στην ανάθεση βαρδιών στο προσωπικό.
\item
    Φόρτωση δεδομένων από τη δοθείσα λίστα φαρμακευτικών προϊόντων στη βάση.
\item
    Εύρεση συνολικού μέσου των αξιολογήσεων κάθε ασθενούς για κάθε νοσηλεία στο ιστορικό του, έναντι μέσης αξιολόγησης κάθε νοσηλείας ξεχωριστά.
\item
    Βελτιστοποίηση λύσης στο ερώτημα \eng{Q08}.
\end{itemize}
